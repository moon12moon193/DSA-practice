# What is a Stack?

A **Stack** is a linear data structure that follows the **LIFO (Last In, First Out)** principle.

The last element inserted into the stack is the first one to be removed.

Example:

```
Push 1
Stack: 1

Push 2
Stack: 1 2

Push 3
Stack: 1 2 3

Pop

Stack: 1 2
```

The last inserted element (`3`) is removed first.

---

# Basic Stack Operations

| Operation | Description | Time Complexity |
|-----------|-------------|-----------------|
| `push()` | Insert an element | O(1) |
| `pop()` | Remove the top element | O(1) |
| `top()` | Access the top element | O(1) |
| `empty()` | Check if the stack is empty | O(1) |
| `size()` | Number of elements | O(1) |

# When to Use a Stack?

A stack is useful when solving problems involving:

- Recently processed elements
- Matching pairs (parentheses, brackets)
- Undo/Redo operations
- Browser history
- Function call management (Call Stack)
- Backtracking
- Expression evaluation
- Monotonic increasing/decreasing sequences
- DFS (Depth First Search)
- Removing adjacent duplicates
- Next Greater/Smaller Element problems
